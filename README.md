- CSS Framework : Tailwind
- State Management : Redux-Toolkit

- Login/signup

- Browsing Page

    - Main Movie (Trailer in bg with title & description)
    - Movie Suggestions - List of Movies
- chatGpt (integrating with chatgpt API's for features like ⬇️)
    - Search bar
    - Movie suggestions
